- **入口与状态管理**：`main.tsx` 挂载 `App.tsx`，通过 `lib/store.ts` (Zustand + persist) 统一管理多模式内容、主题色及图床配置。
- **模式化视图层**：`modes/` 下按业务场景（article, document, card, html）拆分独立工作台，采用懒加载策略优化首屏性能。
- **核心渲染引擎**：`engine/` 模块实现框架无关的 Markdown 解析器 (`markdownParser.ts`)，支持自定义标签（如 `<title>`, `<steps>`）到内联样式 HTML 的转换，确保跨平台兼容性。
- **组件化设计系统**：`engine/editor-components/` 遵循 `{Type}_{Variant}_{ID}` 命名规范（如 `Title_DA01`），封装了20+种高复用视觉组件及其渲染逻辑。
- **工具与导出层**：`lib/` 提供滚动同步、防抖输入处理，以及基于 `jspdf` 和 `modern-screenshot` 的高保真 PDF/图片导出能力。
