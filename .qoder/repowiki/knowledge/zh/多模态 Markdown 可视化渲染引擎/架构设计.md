- **核心分层**：采用 `engine`（纯 TS 解析与组件映射）、`lib`（状态管理与工具逻辑）、`modes`（业务视图层）的三层架构。
- **数据流设计**：基于 `zustand` 实现全局状态持久化，通过 `useEditorDocSync` 处理编辑器防抖同步，确保输入与渲染的一致性。
- **渲染机制**：`engine/utils/markdownParser.ts` 负责将 Markdown 转换为带样式的 HTML，支持自定义标签（如 `<title>`、`<steps>`）；`modes` 层根据场景进行二次加工（如 A4 分页计算、卡片尺寸适配）。
- **导出方案**：集成 `html2canvas` 与 `jspdf`，在 `DocumentMode` 和 `CardMode` 中通过隐藏 DOM 测量高度实现精准分页截图导出。
