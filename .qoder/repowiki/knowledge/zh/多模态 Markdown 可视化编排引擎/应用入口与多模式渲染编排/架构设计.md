- **入口层**：`src/main.tsx` 作为应用启动点，引入 `katex` 样式并挂载根组件到 DOM，启用 `StrictMode`。
- **核心编排层**：`src/App.tsx` 是唯一的根组件，通过 `useStore` 订阅全局状态（如当前模式、Markdown 内容、主题色），并根据 `mode` 状态使用 `Suspense` 和 `lazy` 动态加载对应的业务模式组件（`ArticleMode`, `DocumentMode` 等）。
- **样式层**：`src/index.css` 定义全局 CSS 变量（如 `--accent`）及特定场景（如 A4 文档打印、表格滚动条隐藏）的底层样式规则，支持 Tailwind CSS 导入。
