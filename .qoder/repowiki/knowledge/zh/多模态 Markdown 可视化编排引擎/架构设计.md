- **动态路由编排**：`App.tsx` 作为顶层容器，利用 `React.lazy` 和 `Suspense` 按需加载四种异构模式组件，根据全局 `mode` 状态进行视图切换。
- **全局状态总线**：`lib/store.ts` (Zustand) 充当核心数据枢纽，统一管理各模式的原始内容（Markdown/HTML）、主题配置及用户偏好，并通过 `persist` 中间件实现跨会话的本地持久化。
- **共享设计系统**：所有子模块共用一套基于 Tailwind CSS 的 UI 组件库（`components/ui`）和主题引擎（`engine/composables/useTheme`），确保视觉风格的一致性。
- **统一交互反馈**：通过顶层注入的 `Toast` 和 `Modal` 组件，为底层业务逻辑提供标准化的用户通知与配置入口。
