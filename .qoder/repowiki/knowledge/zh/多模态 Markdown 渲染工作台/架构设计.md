- **动态路由分发**：`App.tsx` 作为顶层编排器，利用 `React.lazy` 根据全局 `mode` 状态异步加载对应的 Mode 组件，实现工作台的按需渲染。
- **统一状态总线**：基于 `zustand` 构建持久化全局 Store，统一管理各模式的独立内容（Markdown/HTML）、主题配置及用户偏好，并通过 `dirty` 标记机制实现示例内容与用户数据的智能同步。
- **引擎与视图解耦**：`engine` 模块提供框架无关的纯 TypeScript 解析与渲染能力（如 Markdown 解析、数学公式处理），通过 `lib` 层的适配逻辑桥接至 React 视图层，确保核心逻辑的可移植性。
- **响应式布局适配**：采用 `ResizeObserver` 实时监测视口宽度，动态调整导航栏功能密度并触发移动端侧边抽屉，实现跨设备的自适应交互体验。
