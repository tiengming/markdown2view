# 后续优化方向

> 更新日期：2026-06-15
> 来源：早期代码审查中的推荐改进项，尚未实施

---

## 一、PWA 离线支持

- 添加 Service Worker + manifest.json
- 使应用可安装、离线可用
- 预计工作量：0.5-1 天

## 二、中文字体本地化

- 当前 article/card 模式的 Google Fonts CSS 在无网络环境下无法使用
- 方案：将 Noto Serif SC / Noto Sans SC 的 CSS + 子集字体 Base64 化后内联到 `src/engine/themes/fonts.css`
- 预计工作量：2-4 小时

## 三、防抖动态调节

- 当前 article/card 模式硬编码 300ms 防抖
- 可根据输入速度自适应调节（快速输入 → 500ms，慢速输入 → 150ms），提升交互感知流畅度
- 预计工作量：1-2 小时
